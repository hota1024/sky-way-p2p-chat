# skyway-p2p-chat

## How to use

### 1. get your SkyWay SDK API key

<https://webrtc.ecl.ntt.com/>

### 2. add the avaiable domain

add domain `localhost` to your application avaiable domains.

### 3. make `.env`

```
NEXT_PUBLIC_SKYWAY_KEY= # your application token here
```

### 4. start app

```
yarn dev
```
